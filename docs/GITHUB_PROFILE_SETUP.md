# Как включить красивый Overview на GitHub

1. Зайди на GitHub → **New repository**.
2. Имя репозитория: **ровно твой ник** — `SeatSinister` (как логин).
3. Сделай репозиторий **публичным**, без README (или с README — потом замени).
4. Создай файл **`README.md`** в корне и вставь туда содержимое из **`GITHUB_PROFILE_README.md`** (без верхней строки-комментария про копирование).
5. Commit в ветку **`main`** (или `master` — тогда в настройках профиля выбери нужную ветку: **Settings → Profile → Profile README**).

Через минуту открой свой профиль — над закреплёнными репозиториями появится этот блок.

**Описание репозитория anti-ctf-tasks** сейчас с опечаткой — в GitHub открой репо → шестерёнка **About** → Description, напиши нормально, например:  
`Два anti-CTF таска на C++/ImGui для курса`  
(так закреплённая карточка не будет выглядеть случайной.)
