# VOID CONFIG — сборка (CTF, ImGui + DirectX 11)

## Требования

- Windows 10/11 x64  
- Visual Studio 2022 (или новее) с рабочей нагрузкой **«Разработка классических приложений на C++»**  
- (Опционально) DirectX SDK June 2010 — только если в `.vcxproj` остались пути `DXSDK`; для типичной сборки достаточно Windows SDK из VS  

## Сборка

1. Откройте `imgui dx11 3.sln` в Visual Studio.  
2. Конфигурация: **Release** (или **Debug**), платформа: **x64**.  
3. **Сборка → Собрать решение** (Ctrl+Shift+B).  

Исполняемый файл: `x64\Release\imgui dx11 3.exe` (или `x64\Debug\...`).

## Флаги компилятора

- Подсистема: **Windows** (без консоли).  
- Точка входа: `wWinMain`.  
- Исходники в UTF-8: `/utf-8` (эмодзи и строки UI).  

## Структура

| Файл | Назначение |
|------|------------|
| `main.cpp` | Окно Win32, DX11, ImGui, CTF-логика, разметка UI |
| `custom.h` / `custom.cpp` | Частицы, фон, свечение, рябь, след курсора, тайпрайтер комбо |
| `globals.h` | Состояние виджетов (`checkbox`, `slider`, `item`, список режимов) |
| `imgui/` | Dear ImGui + бэкенды Win32/DX11 |

## CTF (для организаторов)

Условие выдачи флага: **Enable = ON**, **FOV = 271**, **Mode = Full** (`items[4]`).  
Строки флага и «ложного» сообщения хранятся как XOR `0x5A` в `main.cpp`.
