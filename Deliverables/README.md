## Архивы для сдачи (Windows)

Тексты заданий: [CTF_TASK_STATEMENTS.md](../CTF_TASK_STATEMENTS.md), коротко [ANTIctf.md](../ANTIctf.md).

- `ImGuiVaultFresh_windows.zip`

Шрифт и фон зашиты в exe, отдельные файлы в архив не обязательны.
