# Проверка для преподавателя (спойлеры)

Не публиковать участникам вместе с условием. Использовать для приёмки и демонстрации solve.

## Timing Gate FSM (`timing_gate.exe`)

- Ожидаемая последовательность цифр: **2 → 7 → 1 → 8** (проверяется после нажатия **«Старт сессии»**).
- Между цифрами проверяются **интервалы времени** (мс) между нажатиями; при верной последовательности, но неверных паузах — сценарий «почти верно» / ложный флаг.
- Успех: попап с настоящим `flag{...}`.

## ImGui Vault (`vault_fresh.exe`)

- Есть **ложный PIN** из подсказки и **настоящий PIN** для текущего профиля; набор профилей **ротируется** в зависимости от сценария с `vault.dat` и перезапусков после блокировки.
- Очень бледный текст на **прогресс-баре** несёт часть подсказки.
- Настоящий флаг после верного PIN отображается **в XOR**; ключ для расшифровки подсвечивается в интерфейсе в соответствующей ветке.
