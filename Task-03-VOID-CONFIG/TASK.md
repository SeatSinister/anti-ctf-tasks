# Eclipse Config (VOID CONFIG v2.1) — statement

## Что это
Окно конфигуратора в стиле «чит-меню» (Windows + DirectX 11). Флаг появляется только при **точной** комбинации настроек.

## Как получить флаг
1. Запусти приложение из решения Visual Studio `imgui dx11 3.sln` (собери `Debug/Release`).
2. В интерфейсе «КОНФИГ VOID» выставь:
   - чекбокс/кнопку **Включить**,
   - **Угол обзора (FOV)** в нужное значение,
   - **Режим** = нужный пункт в выпадающем списке.
3. Нажми кнопку **ПРИМЕНИТЬ КОНФИГ**.

## Ложная ветка (обманка)
В интерфейсе может появиться попап «успеха», но с фейковым текстом. Решение засчитывается только при правильной проверке конфигурации.

## Формат флага
`flqg{...}`
