## Deliverables for Teacher (Windows)

**Условия заданий для участников (как в CTF):** [`CTF_TASK_STATEMENTS.md`](../CTF_TASK_STATEMENTS.md) — полная версия с описанием. Укороченный вариант для вставки в Word: [`ANTIctf.md`](../ANTIctf.md).  
**Проверка / спойлеры:** [`TEACHER_VERIFICATION.md`](../TEACHER_VERIFICATION.md) — только для преподавателя.

Two ready-to-run archives are included:

- `ImGuiVaultFresh_windows.zip`
- `RhythmForensics_windows.zip`

### How to run

1. Unzip archive.
2. Run corresponding `.exe`.

### Notes

- If app does not start on a clean machine, install Microsoft Visual C++ Redistributable 2015-2022 (x64).
- Шрифт и фон **вшиты в `.exe`** (Windows `RCDATA`); в архиве достаточно одного исполняемого файла.
