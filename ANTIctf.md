# Условия (коротко)

Формат флага: `flag{...}`

## 1. Timing Gate

Файл: `timing_gate.exe`

Задача: получить настоящий доступ и флаг `flag{...}`.

Подсказка: в программе сказано про фазы и паузы между шагами. Цифры без нужного ритма могут не подойти.

## 2. ImGui Vault

Файл: `vault_fresh.exe`

Задача: открыть сейф по правильной ветке и записать `flag{...}`.

Подсказка: не только крупные буквы; есть еле видимый текст. Смотри ещё что после ошибок и при новом запуске.

## Примечание

В архив достаточно exe + этот файл. Антивирус может ложно срабатывать — тогда сборка из git.
